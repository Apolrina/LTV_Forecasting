This repository contains an end-to-end analytical project focused on forecasting long-term customer revenue in a recurring-payment digital product. The analysis combines retention dynamics, cohort behavior, and probabilistic renewal patterns to estimate expected revenue contribution at the individual user level.

Overview
Recurring revenue businesses rely heavily on customer longevity and renewal consistency. Understanding how engagement evolves over time allows companies to better estimate future cash flow, optimize acquisition strategy, and evaluate customer quality across different signup periods.

The project explores historical billing activity to model how user retention impacts cumulative revenue generation over a one-year horizon.

Objectives
* Prepare and restructure transactional subscription data for longitudinal analysis
* Investigate retention decay and renewal behavior across user cohorts
* Identify temporal patterns affecting customer survival probability
* Build a forecasting framework for expected customer revenue
* Compare projected revenue performance between acquisition cohorts
