Subscription Lifetime Value (LTV) Modeling

This project focuses on analyzing subscription renewal behavior and building a predictive Lifetime Value (LTV) model for a subscription-based business.

The objective is to estimate the expected revenue generated by users over a 12-month horizon by modeling customer retention and subscription renewals.

Project Scope

* Transform raw subscription data into an analysis-ready format
* Explore renewal dynamics, retention trends, and cohort behavior
* Analyze patterns and dependencies affecting subscription longevity
* Build a user-level LTV forecasting model based on historical renewals
* Compare average LTV across different acquisition cohorts

Modeling Assumptions

* Forecast horizon: 12 months
* Fixed subscription price: $29.99/month
* Revenue depends entirely on subscription renewals and retention behavior

Cohort Analysis

The project includes comparative analysis of user cohorts, including:

* Users who started subscriptions in January 2024
* Users who started subscriptions in December 2024

with the goal of identifying differences in retention and projected customer value.
